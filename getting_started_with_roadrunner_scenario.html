开始使用 RoadRunner 场景

创建并回放自动驾驶模拟场景
RoadRunner Scenario 是一个交互式编辑器，可让设计用于仿真和测试自动驾驶系统的场景。放置车辆和路径、定义逻辑并对场景进行参数化。然后可以在编辑器中仿真该场景。从内置车辆参与者中进行选择，或使用 MATLAB ®和 Simulink ®或CARLA设计自己的车辆参与者。
RoadRunner Scenario 支持编辑器内回放以可视化场景并连接到其他模拟器进行协同仿真。场景可以导出到 ASAM OpenSCENARIO®。这些导出的场景可以在任何符合 OpenSCENARIO 的模拟器和播放器中使用，包括 esmini。
RoadRunner API 允许更改场景的参数，以创建许多用于自动化测试的变体。可以使用 API 来自动化工作流程，例如将场景放置在不同场景中、模拟场景和导出。

教程

  • 探索并模拟简单场景
在 RoadRunner Scenario 中探索和仿真简单的预构建场景，以了解场景设计的基础知识。
  • 设计车道跟随场景
设计一个场景，其中车辆沿着车道行驶并改变速度。
  • 设计变道场景
设计一个车辆在不使用预定义路径的情况下改变车道的场景。
  • 设计车道转弯场景
设计一个场景，其中车辆在车道内从一侧转向另一侧。
  • 设计路径跟随场景
设计一个场景，其中车辆遵循预定义的路径。
