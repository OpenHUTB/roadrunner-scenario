设置 Carla 进行协同仿真

安装 Carla

从 GitHub® 发布网站下载 Carla。将发布版本解压到一个文件夹中，该文件夹成为 CARLA 安装目录 CARLAInstall。
笔记
  • RoadRunner 场景仅使用 CARLA 0.9.12 版本进行了测试。
  • CARLA版本 0.9.12 需要 Python® 3.7。
生成 Carla 协同仿真桥
RoadRunner 场景和 Carla 协同仿真使用 Carla 协同仿真桥进行同步和通信。有关协同仿真桥的信息，请参阅RoadRunner 场景和 Carla 协同仿真概述。
请按照以下步骤生成协同仿真桥：
1. 安装 Python 3.7。
2. 将 Python 添加到您的路径中。有关如何在 Microsoft ® Windows ®中执行此操作的信息，请参阅Python 文档中的 https://docs.python.org/3/using/windows.html#excursus-setting-environment-variables 。
3. 使用这些命令更新pip并安装psutil 软件包。
python -m pip install --install pip
python -m pip install psutil
4. 运行 RoadRunnerInstall\bin\win64\Tools\CARLA\examples\setup.bat，其中 RoadRunnerInstall 是安装 RoadRunner Scenario 的文件夹。

笔记

如果在 Program Files 目录中安装 RoadRunner Scenario，则必须以管理员身份编辑 BAT 文件。
配置协同仿真属性
可以选择修改 RoadRunner Scenario 和 CARLA 之间的协同仿真属性。协同仿真属性包括场景仿真引擎 (SSE) 的时序步长分辨率和网络位置。有关如何配置协同仿真属性的信息，请参阅仿真配置。
