RoadRunner Scenario

创建并回放自动驾驶仿真情景
RoadRunner Scenario 是一个交互式编辑器，可让设计用于模拟和测试自动驾驶系统的情景。放置车辆和路径、定义逻辑并对情景进行参数化。然后可以在编辑器中仿真该情景。从内置车辆参与者中进行选择，或使用 MATLAB ® 和 Simulink ®或CARLA 设计自己的车辆参与者。
RoadRunner Scenario 支持编辑器内回放以可视化情景并连接到其他模拟器进行协同仿真。情景可以导出到 ASAM OpenSCENARIO®。这些导出的情景可以在任何符合 OpenSCENARIO 的模拟器和播放器中使用，包括 esmini。
RoadRunner API 允许您更改情景的参数，以创建许多用于自动化测试的变体。可以使用 API 来自动化工作流程，例如将情景放置在不同场景中、仿真情景和导出。
