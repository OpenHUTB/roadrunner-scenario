导入情景数据

从 ASAM OpenSCENARIO® 导入情景轨迹并导入自定义参与者网格
可以从 ASAM OpenSCENARIO V1.0 导入场景轨迹数据以及自定义参与者网格。可以使用导入的场景数据来设计和修改现实情景。
要导入情景文件，请从RoadRunner 场景工具栏中选择文件，然后选择 导入。或者，您可以使用MATLAB ®函数 importScenario 或 gRPC ® API 的 Import 方法以编程方式导入场景文件。

主题

  • 从 ASAM OpenSCENARIO 文件导入轨迹
从 ASAM OpenSCENARIO 1.0 文件格式定义的场景导入轨迹。
  • 导入自定义车辆网格
定义自定义车辆 FBX®网格并将其导入RoadRunner Scenario 中。
  • 导入自定义角色网格
定义自定义角色 FBX 网格并将其导入RoadRunner Scenario 中。
  • 从 CSV 文件导入轨迹
导入以 CSV 文件格式定义的轨迹数据。
